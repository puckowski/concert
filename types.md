# Types

## Primitive types

- int
- long
- double
- string
- mutex

## Properties

|Type   |Width in bits |
|:------|:-------------|
|int    |at least 32   |
|long   |at least 64   |
|double |at least 64   |

## Declaration

int declaration examples:

```cpp
int x;
int y = 0;
int z = 5 + y;
```

long declaration examples:

```cpp
long l;
long m = 0;
long n = 5 + m;
```

double declaration examples:

```cpp
double d;
double pi = 3.14;
double tenth = 1.0 / 10.0;
```

string declaration examples:

```cpp
string s;
string partialWelcome = "Hello";
string fullWelcome = partialWelcome + ", world!";
```

mutex declaration examples:

```cpp
mutex m;
mutex lock;
```
